# buscadorDinamico

**buscadorDinamico** is a lightweight, dependency-free JavaScript class (`search`) that adds powerful dynamic search capabilities to any web page. Include a single script tag and you get real-time filtering, automatic pagination, and a built-in cache — all with zero framework requirements.

## Key features

<CardGroup cols={2}>
  <Card title="Three data modes" icon="database">
    Use a local JavaScript array, read existing DOM elements, or fetch results from a server via AJAX — all with the same API.
  </Card>
  <Card title="Real-time search" icon="magnifying-glass">
    Input events are debounced at 500 ms, so the DOM is updated only after the user stops typing.
  </Card>
  <Card title="Built-in pagination" icon="list-ol">
    Configure `itemsPerPage` and the library automatically calculates page count, slice indices, and renders navigation buttons.
  </Card>
  <Card title="Unified cache" icon="bolt">
    Both local and server results are cached for 5 minutes using a `Map`-based store, keyed by search term and page number.
  </Card>
  <Card title="Custom rendering" icon="paintbrush">
    Override `renderizarItem` with your own function to control exactly how each search result is displayed.
  </Card>
  <Card title="Auto-generated HTML" icon="wand-magic-sparkles">
    Missing input, results body, or pagination footer? The class creates them automatically so your markup stays minimal.
  </Card>
</CardGroup>

## How it works

The `search` class reads from a CSS selector you provide (`element`), finds or creates three child zones — a search input header, a scrollable results body, and a pagination footer — then wires up all event listeners and renders the initial result set.

```
┌─────────────────────────────┐
│  .app-search  (container)   │
│  ┌───────────────────────┐  │
│  │ <search .input-search>│  │  ← label + text input
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ <main .items-search>  │  │  ← rendered result items
│  └───────────────────────┘  │
│  ┌───────────────────────┐  │
│  │ <footer .index-search>│  │  ← pagination buttons
│  └───────────────────────┘  │
└─────────────────────────────┘
```
